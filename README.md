## PKM/Obsidian Essentials

This vault contains a class I'm developing.

It is also a "proof of concept" of a way to use <span style="color:cyan;">Obsidian</span> to facilitate teaching.

#### Top level

This README and the course outline.

#### Chapters

The lecture, presentation and any othermaterials you with to distribute to your class. 

#### Assignments

Instructions and related material for student exercises.

#### Submissions

This is the folder that students will submit for grading. The "Teacher's Edition" contains a seperate folder in Submissions for each student and a top-level moc documenting grades, progress, etc.

#### html

The html output of Advanced Slides

#### Resources

Images, pdfs, video, etc.
