## Chapter 1 Assignments

1. Install Obsidian
2. Make some notes
3. Explore the settings and plugins
4. Install some plugins that seem interesting
5. Uninstall plugins for now unless you are sure you need them
6. Create 1-3 notes in the submissions folder using the following elements
	- Headers
	- Emphasis
	- Links
	- Lists
	- Tags
	- Blockquotes
	- Images
	- Backlinks

Bonus: Include a table in your notes.
